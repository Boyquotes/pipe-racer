Pipe Racer Roadmap

Version 1.1

Replay System

  • Runs can be saved as a replay
  • PBs will be uploaded to server
  • Replays can be viewed in-game
  • Replays can be raced against (Ghosts)

    • Social Features:

    • Notifications for beaten records
    • Firends
    • View uploaded maps by players

Version 1.2

Medals

  • Tracks must be validated before upload
  • Validation time will be added as a medal
  • Gold/Silver/Bronze will be automatic
  • Medals can be collected
  • New Map File Format

    • Social Features:

    • View medals collected

Editor extension

  • Better snapping implementation
  • Maybe sprint mode (?)

    • Ask for rating after exiting new map
